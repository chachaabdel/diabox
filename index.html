Diabox Examples (download)

Diabox is another modal window lightbox thing built on the Mootools JS framework. It provides support for the following content types and is extremely customizable.

  • Images
  • Local HTML (via ajax)
  • Remote Pages
  • PDF's
  • Powerpoints
  • TIFF's
  • Youtube Videos
  • Vimeo Videos
  • Elements
  • Inline Content
  • Raw Text
